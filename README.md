# 🚀 ALIGNA — Hire Top Talent Faster With AI Precision

ALIGNA is an AI-powered talent scouting and engagement agent that helps recruiters discover, assess, and rank candidates instantly.

## ✨ Features

- **JD Parser**: Automatically extract requirements from text or PDF.
- **Matching Engine**: Score candidates based on skills, experience, and location.
- **Interest Simulation**: AI-driven outreach to gauge candidate interest.
- **Ranked Shortlist**: Receive a prioritized list of "Hot Leads" ready for action.
- **Dashboard**: Professional glassmorphism UI for managing the pipeline.

## 🛠️ Tech Stack

- **Frontend**: Next.js 16, TypeScript, Tailwind CSS v4.
- **Backend**: FastAPI (Python), Vercel Serverless.
- **AI**: Gemini 2.0 Flash & Groq (Llama 3.3).
- **DB**: MongoDB Atlas.

## 🚀 Getting Started

Check out [LOCAL_SETUP.md](./LOCAL_SETUP.md) for full instructions.

1. Install dependencies: `npm install`
2. Set up `.env.local` with API keys.
3. Run backend: `uvicorn api.index:app --reload`
4. Run frontend: `npm run dev`
